# Reinforcement Learning Project – Grid World

## 📌 فكرة المشروع

المشروع ده عبارة عن تطبيق عملي على Reinforcement Learning باستخدام بيئة بسيطة اسمها Grid World.
الهدف إننا نخلي Agent يتعلم يوصل لنقطة معينة (Goal) بأفضل طريق ممكن مع تجنب العقبات.

أنا استخدمت خوارزميتين:

* Q-Learning
* SARSA

والفكرة الأساسية كانت إني أقارن بينهم وأفهم الفرق بينهم بشكل عملي مش نظري بس.

---

## 🧠 البيئة (Environment)

البيئة عبارة عن Grid 3x4:

* نقطة بداية (Start)
* نقطة هدف (Goal)
* Obstacles

الـ Agent يقدر يتحرك:

* Up
* Down
* Left
* Right

### نظام الـ Rewards:

* +10 عند الوصول للهدف
* -10 عند الاصطدام بعقبة
* -1 لكل خطوة (علشان يتعلم ياخد أقصر طريق)

---

## ⚙️ الخوارزميات

### 🔵 Q-Learning

خوارزمية Off-policy
بتفترض إن أفضل حركة دايمًا هتتاخد في المستقبل
وده بيخليها أسرع في التعلم لكن أحيانًا غير مستقرة

---

### 🟢 SARSA

خوارزمية On-policy
بتتعلم بناءً على الفعل الفعلي اللي حصل
وده بيخليها أهدى وأكثر استقرارًا

---

## 📊 النتائج

من خلال التجارب:

* Q-Learning:

  * اتعلم بسرعة
  * لكن فيه fluctuations (عدم استقرار)

* SARSA:

  * أبطأ شوية
  * لكن stable بشكل واضح

وده كان واضح جدًا في Learning Curve
